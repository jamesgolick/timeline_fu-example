h1. TimelineFu Example Application

This is a sample application demonstrating the use of TimelineFu.

This is a social application:  People have friends, described through relationships.

Relationships create timeline events.

See the accompanying article at giraffesoft's blog:  "How to Use TimelineFu: Building a timeline for a social application":http://giraffesoft.ca/blog/2009/02/25/how-to-use-timelinefu-building-a-timeline-for-a-social-application.html
